## Responsive Behavior

### Auto-scroll
- Automatically scrolls to bottom when new messages arrive
- Smooth animation using `animateScrollToItem()`

### Message Bubbles
- **Max Width**: 300dp (75% of screen width on most devices)
- **Min Height**: Wraps content
- **Padding**: 12dp internal, 4dp between messages
- **Alignment**: 
  - User: End (right)
  - AI: Start (left)

### Streaming Metrics
- **Updates**: Every 30 frames (~1 second)
- **Display**: `{fps} FPS | {latency}ms`
- **Position**: Right side of status bar

## Accessibility

### Content Descriptions
- All interactive elements have descriptive labels
- Icons include text alternatives
- Status indicators readable by screen readers

### Color Contrast
- Text on colored backgrounds meets WCAG AA standards
- White text on status colors for maximum visibility
- High contrast mode supported via system settings
